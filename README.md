## Require
    * Tomcat
    * Maven

## How to run:
    1. Build project:
        mvn clean install
    2. Copy chat-be.war (chat-be folder) or to tomcat-folder/webapps
    3. Start tomcat: 
        In tomcat-folder/bin run startup.bat (startup.sh)
        To stop run shutdown.bat (shutdown.sh)
